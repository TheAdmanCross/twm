# Trading With Matt Web Starter

Premium Astro + React starter for a cinematic personal-brand landing page with Decap CMS editing.

## Quick start

```bash
npm install
npm run dev
```

## Build output

- Astro source entry: `src/pages/index.astro`
- Static deploy output: `dist/index.html`
- CMS editor: `/admin`
